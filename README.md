Розробка бази даних компанії-продавця будівельних матеріалів

Опис: Це навчальний веб-проєкт, у якому реалізовано підключення бази даних **MySQL** до веб-сайту за допомогою фреймворку **Flask**.  
Через веб-інтерфейс користувач може переглядати та керувати даними бази даних компанії-продавця будівельних матеріалів.

Використані технології:
- Python
- Flask
- MySQL
- mysql-connector-python
- HTML / CSS (шаблони Flask)

Як запустити проєкт:
1. Створити віртуальне середовище: python -m venv venv
2. Активувати віртуальне середовище: venv\Scripts\activate
3. Встановити залежності: pip install -r requirements.txt
4. Запустити проєкт: python app.py
5. Відкрити сайт у браузері: http://127.0.0.1:5000
